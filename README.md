# progetto-scuola
progetto di scuola che simula una applicazione di ebike che stampa in una tabella le bici esistenti e ti consente di scegliere quale usare che, in base alla diasponibilità, ti consente di inserire la distanza che si deve percorrere finendo col calcolo del tempo sul quale si basa il pagamento ed aggiorna l'autonomia della bici usata
